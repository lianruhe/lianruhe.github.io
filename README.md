# lianruhe.github.io

Powered by [Hexo](https://hexo.io/). Theme by [BlueLake](https://github.com/chaooo/hexo-theme-BlueLake).

## Usage

```bash
  # 新建一篇文章
  $ hexo new [layout] <title>

  # 生成静态文件
  $ hexo g/generate -w

  # 启动服务器
  $ hexo server -p <4000> --debug

  # 清除缓存文件
  $ hexo clean

  # 发布
  $ npm run publish
```
